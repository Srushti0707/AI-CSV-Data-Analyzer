# AI-Powered CSV Data Analyzer

An intelligent web-based dashboard that simplifies data cleaning, exploratory data analysis (EDA), and AI-driven insight generation from CSV datasets. Built with Python, Streamlit, Pandas, Google Gemini AI, and SQLite, this application enables users to upload datasets, perform preprocessing, analyze key statistics, and receive natural-language insights generated by AI.


## Overview

The AI-Powered CSV Data Analyzer is designed to help users quickly understand and explore datasets without requiring extensive data analysis expertise. The application combines traditional data preprocessing techniques with generative AI to provide meaningful insights and recommendations from uploaded CSV files.



## Features

### CSV File Upload

Upload any standard CSV dataset and begin analyzing data through an interactive web interface.

### Data Cleaning

Handle missing values using multiple preprocessing strategies:

* Drop rows containing missing values
* Fill numerical columns using mean values
* Fill categorical columns using mode values

### Exploratory Data Analysis

Automatically generates descriptive statistics for numerical columns, including:

* Count
* Mean
* Standard Deviation
* Minimum Value
* Maximum Value
* Quartiles

### AI-Powered Insights

Utilizes Google Gemini 2.5 Flash to analyze dataset summaries and sample records, generating:

* Key trends and patterns
* Potential anomalies
* Data-driven observations
* Suggested analytical questions

### Session History Management

Stores completed analyses in a local SQLite database, allowing users to:

* View previous analysis sessions
* Reload historical insights
* Maintain a persistent analysis history

### Interactive User Interface

Built with Streamlit to provide:

* Responsive layout
* Interactive controls
* Organized data presentation
* Easy navigation through analysis history

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Srushti0707/AI-CSV-Data-Analyzer.git
cd AI-CSV-Data-Analyzer
```

### 2. Install Dependencies

Ensure Python 3.9 or later is installed.

```bash
pip install streamlit pandas google-genai python-dotenv
```

### 3. Configure Environment Variables

Create a `.env` file in the project root directory and add your Gemini API key:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### 4. Run the Application

Launch the Streamlit dashboard:

```bash
streamlit run app.py
```

## System Workflow

1. User uploads a CSV file.
2. Dataset is cleaned using the selected preprocessing method.
3. Statistical summaries are generated using Pandas.
4. Dataset information and sample records are analyzed.
5. Gemini AI generates insights, observations, and recommendations.
6. Results are displayed through the Streamlit dashboard.
7. Analysis history is stored locally and can be accessed later.



## Security Considerations

* API keys are stored securely using environment variables.
* The `.env` file should never be committed to GitHub.
* Analysis history remains stored locally on the user's machine.
* No sensitive credentials are hardcoded into the application.
